# AWS Practice Exam App

Это небольшое приложение для подготовки к экзамену **AWS Certified Cloud Practitioner (CLF-C02)**. Интерфейс создан с помощью Streamlit.

##  Что входит

- `app.py` — основной код приложения
- `requirements.txt` — список зависимостей
- `install_env.bat` — скрипт для установки окружения и зависимостей
- `start_app.bat` — скрипт для запуска приложения
- Папка `exams/` — содержит тренировочные тесты

##  Быстрый старт

### 1. Установи зависимости

Дважды кликни на `install_env.bat`.  
Он создаст виртуальное окружение и установит всё необходимое.

### 2. Запусти приложение

Дважды кликни на `start_app.bat`.  
Откроется браузер с веб-интерфейсом приложения.

## Требования

- Установленный [Python 3.8+](https://www.python.org/downloads/)
- Windows (используются `.bat`-файлы)
